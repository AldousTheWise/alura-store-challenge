## Análisis de Datos – Challenges Data Science (Latam)

### Descripción general

Este repositorio contiene el desarrollo de dos challenges de Data Science, realizados en Python utilizando Google Colab como entorno principal de trabajo.
El objetivo de los proyectos es aplicar técnicas de análisis exploratorio de datos, visualización y extracción de insights para apoyar la toma de decisiones basada en datos.

Los análisis se centran en datos de ventas, comportamiento de clientes y distribución geográfica, con énfasis en la claridad del razonamiento y la correcta interpretación de resultados.
