# Análisis de Ventas – Alura Store

## Descripción del proyecto

Este proyecto tiene como objetivo analizar el desempeño de las cuatro tiendas de la cadena Alura Store, con el fin de recomendar cuál de ellas debería vender el Sr. Juan para iniciar un nuevo emprendimiento.
El análisis se basa en datos de ventas reales y considera métricas financieras, comerciales y de satisfacción del cliente, apoyándose en visualizaciones para facilitar la interpretación de los resultados.
El desarrollo del proyecto se realizó en **Google Colab**, utilizando **Python** y **bibliotecas** de análisis de datos.

### **Objetivos del análisis**

- Calcular los ingresos totales de cada tienda.
- Analizar las categorías de productos más y menos vendidas.
- Evaluar la satisfacción de los clientes a través de las calificaciones promedio.
- Identificar los productos más y menos vendidos por tienda.
- Calcular el costo de envío promedio por tienda.
- Explorar la distribución geográfica de las ventas mediante coordenadas de latitud y longitud.
- Emitir una recomendación final basada en los resultados obtenidos.

## Tecnologías y bibliotecas utilizadas

- Python 3
- Pandas
- Matplotlib
- Folium
- Google Colab

## Estructura del proyecto

El repositorio contiene:

1. Un notebook (.ipynb) con:
   - Carga y limpieza de datos
   - Análisis exploratorio
   - Generación de tablas y métricas
   - Visualizaciones gráficas
   - Mapas interactivos
   - Informe final con conclusiones

\*_Los datos utilizados se cargan directamente desde los repositorios oficiales del desafío mediante URLs públicas._

## Visualizaciones incluidas

- Gráficos de barras para ingresos y categorías de productos.
- Gráficos comparativos de calificaciones promedio.
- Gráficos de productos más y menos vendidos.
- Gráficos de participación porcentual de ingresos.
- Mapas interactivos y mapas de calor para analizar la distribución geográfica de las ventas.

## Resultados y conclusión

A partir del análisis de ingresos, volumen de ventas, satisfacción del cliente, costos de envío y distribución geográfica, se presenta una recomendación fundamentada sobre cuál tienda representa la menor eficiencia relativa dentro de la cadena y, por lo tanto, es la candidata más adecuada para ser vendida.

La decisión final se encuentra documentada en el informe incluido dentro del notebook.
